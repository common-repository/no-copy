== Description ==
No Copy plugin is a upload and activate plugin which prevents visitors the context menu (right click) on your blog. This should prevent plagirism by most users. However there are many means to copy a site without using the context menu.

== Installation ==

1. Upload `no-copy` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to view the content in a blog where No Copy is installed? =

Go to 'View' -> 'Page Source' in most browsers. Ctrl+U also works in most Mozilla based browsers, if not all.
